# Ubiquitous Language

プロジェクト内で共通認識を得るためのユビキタス言語を辞書化するリポジトリ。

## contribute

1. gitのブランチを切る
  * `$ git switch -c feat/word_name`
2. ローカルサーバでプレビューを開始する
  * `$ docker-compose up`
  * [http://localhost:3000](http://localhost:3000)
3. ドキュメントを変更する
  * docsifyのドキュメントは[ここ](https://docsify.now.sh)
4. master宛にPRを出して全員のレビューを通過したらマージする
